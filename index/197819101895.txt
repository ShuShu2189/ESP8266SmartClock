下面是一个简单的 HTML 页面示例，用于实现你所描述的功能：

```html
<!DOCTYPE html>
<html>
<head>
    <title>登录页面</title>
</head>
<body>
    <div>
        <h1>登录</h1>
        <form id="loginForm">
            <label for="username">账号：</label>
            <input type="text" id="username" name="username" required><br><br>
            
            <label for="password">密码：</label>
            <input type="password" id="password" name="password" required><br><br>
            
            <button type="button" onclick="saveCredentials()">保存</button>
        </form>
    </div>

    <script>
        function saveCredentials() {
            var username = document.getElementById('username').value;
            var password = document.getElementById('password').value;
            
            // 在这里将账号和密码发送到数据库中保存
            // 可以使用 AJAX 或者其他后端技术来实现这一步
            
            alert('账号：' + username + '\n密码：' + password + '\n已保存至数据库');
        }
    </script>
</body>
</html>
```

在这个示例中，通过 HTML 的 `<input>` 元素和 `<form>` 元素来创建账号和密码的输入框，并且使用了一个 JavaScript 函数 `saveCredentials()` 来获取输入框中的值，并保存至数据库。在这个例子中，为了方便演示，保存操作只是简单的弹出一个提示框，并未实际发送到数据库。

你可以将这段代码保存为一个 `.html` 文件，并在浏览器中打开，即可看到一个简单的包含账号密码输入框的登录页面。
