# 部署指南
1. Gerber文件拿去打板
2. ino文件烧到esp8266里面
3. 根据原理图把各种原件焊在板子上，然后插到对应的插槽

# 使用说明
手机搜索热点，默认名称Smart WiFi Clock，密码是12345678，在ino文件里面WiFiManager.autoConnect("Smart WiFi Clock", "12345678")那里，有需要的可以自己改。打开浏览器输入192.168.4.1，连接后进入后台管理页面，设置需要的选项，保存之后即可正常使用。

# 常见问题
Q：屏幕闪烁、屏幕暗、声音小？
A：通病，多充回电，或者多并联两个喇叭

Q：WiFi找不到、连不上？
A：暂不支持WiFi6，名字中有特殊字符可能会卡住，重启一下就好了（试了无数次只遇到了一次，有空的话可以加个特殊字符库，应该能修复）

Q：计时不准？
A：内部时钟电池没电，买块新的CR2032换上（没有特殊要求，主播换的是南孚传应的石墨烯纽扣电池）

Q：屏幕乱码、卡住、显示黑块、白块？
A：对表失败，可能是计时网站出问题了，可以重启，如果重启之后还有这种问题，可以尝试更换对表网站，目前只能在代码里面改，不过这个问题极难触发，一般是没连上网才会出现，就先不修了

Q：喇叭一直尖叫？
A：短路，检查引脚是否存在连锡
